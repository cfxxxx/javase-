## 排序的分类

1. 内部排序：
   
   指将需要处理的所有数据都加载到内部存储器中进行排序，包括（交换式排序法，选择排序法和插入式排序法）
2. 外部排序法：
   
   数据量过大，无法全部加载到内存中，需要借助外部存储进行排序，包括（合并排序法和直接合并排序法）

## 冒泡排序

```java
// 设给arr数组长度为n的数组排序
for (int i = 0; i < n - 1; i++){
  for (int j = 0; j < n - i - 1; j++){
    if(arr[j] > arr[j + 1]){
      temp = arr[j];
      arr[j] = arr[j + 1];
      arr[j + 1] = temp;
    }
  }
}
```
