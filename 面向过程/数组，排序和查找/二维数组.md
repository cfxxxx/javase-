## 使用方式1：动态初始化

语法：类型[][] 数组名 = new 类型\[大小][大小]

## 使用方式2：动态初始化

1. 先声明： 类型 数组名[][];
2. 再定义（开辟空间） 数组名 = new 类型\[大小][大小]
3. 赋值（有默认值，比如int类型的就是0）

## 使用方式3：动态初始化——列数不确定

```java
int arr[][] = new int[3][];
for(int i = 0; i < 3; i++){
  arr[i] = new int[i + 1];
}
```

## 使用方式4：静态初始化

定义：类型 数组名[][] = {{值1，值2...},{值1，值2...},{值1，值2...}}

## 二维数组使用细节和注意事项

1. 一维数组的声明方式有：
   
   int[] x 或者 int x[]
2. 二维数组的声明方式有：
   
   int[][] y 或者 int[] y[] 或者 int y[][]
3. 二维数组实际上是由多个一维数组组成的，他的各个一维数组的长度可以相同，也可以不同
