## 取模运算%

余数的正负始终由被模数的正负决定

```java
10 % 3 == 1;
-10 % 3 == -1;
10 % -3 == 1;
-10 % -3 == -1;
```

## 除法运算/

```java
10 / 4 == 2；
10.0 / 4 == 2.5；
double d = 10 / 4;  //由于10和4都是整型，所以结果应为2，
                    //但d是浮点型，所以d的值应为2.0
```

## 自增运算++

前++： ++i先自增后赋值

后++： ++i先赋值后自增

```java
int i1 = 8;
int i2 = 8;
int j = ++i1;  //j的结果是9，i1的值是9
int k = i2++;  //k的结果是8，i2的值是9
```

> 看一道面试题
> 
> ```java
> int i = 1;
> i = i++;
> //输出i的结果为1
> //(1) temp = i    (2) i = i + 1   (3) i = temp
> 
> 
> int i = 1;
> i = ++i;
> //输出i的结果为2
> //(1) i = i + 1   (2) temp = i    (3) i = temp
> ```
